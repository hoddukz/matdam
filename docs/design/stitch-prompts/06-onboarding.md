# Stitch Prompt: Onboarding Flow

> 소스: implementation-plan.md Step 12 + master-plan §6.6

---

## 페이지 정보

- **URL:** `/onboarding`
- **용도:** 가입 직후 식단 제한 + 난이도 선택. 개인화 추천의 기반.

## 플로우 (3 스텝)

### Step 1: 환영 + 식단 제한

```
┌─────────────────────────────────────┐
│ [MatDam 로고]                        │
├─────────────────────────────────────┤
│                                     │
│  Welcome to MatDam! 🏺              │
│                                     │
│  Let's personalize your             │
│  Korean cooking journey.            │
│                                     │
│  Do you have any dietary            │
│  restrictions?                      │
│                                     │
│  (선택하지 않아도 됨 — 다중 선택 가능)  │
│                                     │
│  ┌──────────┐ ┌──────────┐         │
│  │ 🌱 Vegan │ │ 🕌 Halal │         │
│  └──────────┘ └──────────┘         │
│  ┌──────────────┐ ┌──────────┐     │
│  │ 🌾 Gluten-   │ │ 🥜 Nut-  │     │
│  │    free      │ │    free  │     │
│  └──────────────┘ └──────────┘     │
│  ┌──────────────┐ ┌──────────┐     │
│  │ 🐟 Pescatarian│ │ 🌶 Spicy │     │
│  │              │ │   OK     │     │
│  └──────────────┘ └──────────┘     │
│                                     │
│  [Next →]              [Skip all]  │
│                                     │
└─────────────────────────────────────┘
```

- 큰 카드 버튼 (토글 선택)
- 선택된 카드는 골드 보더 + 체크마크
- 아무것도 안 골라도 Next 가능

### Step 2: 난이도

```
┌─────────────────────────────────────┐
│                                     │
│  How experienced are you            │
│  with Korean cooking?               │
│                                     │
│  ┌─────────────────────────────┐    │
│  │ 🔰 Beginner                 │    │
│  │ "I'm just starting out"     │    │
│  └─────────────────────────────┘    │
│  ┌─────────────────────────────┐    │
│  │ 🍳 Intermediate              │    │
│  │ "I've made a few dishes"     │    │
│  └─────────────────────────────┘    │
│  ┌─────────────────────────────┐    │
│  │ 👨‍🍳 Advanced                 │    │
│  │ "Korean food is my thing"    │    │
│  └─────────────────────────────┘    │
│                                     │
│  [← Back]  [Next →]   [Skip all]  │
│                                     │
└─────────────────────────────────────┘
```

- 단일 선택 (라디오 스타일이지만 카드 형태)
- 각 카드에 아이콘 + 짧은 설명

### Step 3: 완료

```
┌─────────────────────────────────────┐
│                                     │
│  You're all set! 🎉                │
│                                     │
│  We'll show you recipes that        │
│  match your preferences.            │
│  You can change these anytime       │
│  in Settings.                       │
│                                     │
│  [Start Exploring →]                │
│                                     │
└─────────────────────────────────────┘
```

- 완료 메시지 + Explore 페이지로 이동하는 CTA

## 핵심 포인트

1. **스킵 가능** — "Skip all" 항상 보이게. 온보딩이 가입 장벽이 되면 안 됨
2. **3 스텝 이내** — 길면 이탈. 30초 안에 끝나야 함
3. **선택이 부담 없어야 함** — "나중에 바꿀 수 있어요"를 강조
4. **식단 제한이 1순위** — 이게 있어야 Dietary Filter가 의미 있음
